# Memory Bank — ProjeHaritası

Bu klasör AI asistanların oturumlar arası sürekliliği koruması için kullanılır.

## Nasıl Çalışır

Her yeni göreve başlamadan önce AI bu klasördeki tüm dosyaları okumalıdır.
Değişiklik yapıldıkça ilgili dosyalar anında güncellenmelidir.

## Dosyalar

| Dosya | İçerik |
|---|---|
| `AI_GUIDELINES.md` | AI davranış kuralları (bu proje için) |
| `projectBrief.md` | Proje kapsamı, hedefler, temel gereksinimler |
| `activeContext.md` | Aktif odak, son değişiklikler, sonraki adımlar |
| `progress.md` | Ne çalışıyor, ne kaldı, bilinen sorunlar |
| `systemPatterns.md` | Mimari kararlar, tasarım desenleri, modül ilişkileri |
| `techContext.md` | Teknolojiler, bağımlılıklar, geliştirme kurulumu |
